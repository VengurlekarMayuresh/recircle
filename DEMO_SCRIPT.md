### STEP 8 — Start Bargaining (AI NEGOTIATION DEMO)
**Click:** "Start Bargaining" on the material

This is another wow moment:
- A bargain session opens with WhatsApp-style chat UI
- **Make an offer** — e.g., ₹2000 (way below asking price of ₹5000)
- The **Bargain Agent** (Gemini) responds on behalf of the seller
- It uses tactics: anchoring, value justification, small concessions
- **It never reveals the floor price** (₹3000)
- Make another offer — e.g., ₹3500 → agent counters with maybe ₹4200
- Keep going → eventually agree on a price
- Say: *"This AI agent is negotiating like a real shopkeeper. It knows the market average, the demand level, and the seller's floor price — but it never reveals the floor price. Each concession gets smaller."*

If the bargain reaches agreement, a transaction is automatically created.

## TALKING POINTS — What to Tell the Judges

Use these when the judges ask questions or when you're explaining between screens.

---

### The Problem We're Solving
- India generates **62 million tonnes** of waste annually — most of it ends up in landfills
- Reusable materials like bricks, wood, metal, furniture get thrown away because there's no easy way to find someone who needs them
- Small businesses, NGOs, and individuals don't have a platform to exchange surplus materials locally
- There's no transparency in environmental impact — companies can't track their circular economy contributions

### Our Solution in One Line
> "ReCircle is like OLX meets sustainability — but with AI agents that do the heavy lifting."

### What Makes Us Different (Top 5 — Memorize These)
1. **AI-first listing** — Upload a photo, Gemini Vision AI fills everything. Zero friction for the seller.
2. **AI Bargain Agent** — Negotiates price automatically like a real shopkeeper. Seller sets floor price, AI handles the rest. Never reveals the floor price.
3. **Scout Agent matches supply with demand** — Every new listing is auto-analyzed and matched against Want Board requests. No manual searching needed.
4. **Volunteer courier network** — Community members deliver materials for free, earn Green Points. Identity verified with selfie + ID.
5. **Real impact tracking** — Every exchange updates CO₂ saved, water saved, rupees saved. Dashboard pulls from real DB. ESG reports auto-generated for businesses.

### The 4 AI Agents (If Asked "What AI Did You Use?")
1. **Scout Agent** — Runs when a new material is listed. Computes Reuse Potential Score (0-100), generates reuse ideas, routes material (reuse/repair/recycle/dispose), matches with Want Board demand. Uses Gemini 2.5 Flash.
2. **Bargain Agent** — Negotiates on behalf of the seller in real-time chat. Uses tactics like anchoring, concession laddering, urgency, value justification. Adapts based on buyer's trust score and market data. Uses Gemini.
3. **Advisor Agent** — Chat assistant with 8 tools. Can search the marketplace DB, estimate impact, suggest reuse ideas, find repair hubs, forecast demand, suggest cross-industry waste symbiosis. It's agentic — it decides which tool to call. Uses Gemini with function calling.
4. **Sentinel Agent** — Scans listings for fraud, unrealistic pricing, suspicious descriptions. Auto-flags for admin review.

### Circular Economy Angle (If Asked "How Does This Help the Environment?")
- Every material reused = less mining, less manufacturing, less landfill
- We track: **CO₂ saved** (using CPCB emission factors per material category), **water saved**, **rupees saved** (vs buying new), **kg diverted from landfill**
- Dashboard shows equivalencies: trees planted, households powered, water buckets saved
- Sankey diagram visualizes the full material lifecycle: Source → Category → Destination
- ESG report page lets businesses export their sustainability data for CSR compliance

### Gamification (If Asked "How Do You Drive User Engagement?")
- **Green Points** — earned for listing materials (+10), completing exchanges (+25), volunteer deliveries (+50), getting reviews (+5)
- **Levels** — Seedling → Sprout → Sapling → Tree → Forest (based on Green Points)
- **Badges** — First Exchange, Eco Warrior, Community Champion, etc.
- **Leaderboard** — Top Impact, Top Suppliers, Most Trusted
- **Trust Score** — 0 to 100, based on verification level, completed transactions, reviews, response rate

### Multi-Stakeholder Platform (If Asked "Who Uses This?")
- **Businesses** — List surplus construction materials, packaging, industrial waste. Get ESG reports.
- **Individuals** — Buy/sell/donate furniture, electronics, textiles. Save money.
- **NGOs** — Post demand on Want Board for materials they need (building supplies, clothing, etc.)
- **Volunteer Couriers** — Community members who deliver for free, earn points and badges
- **Transporters** — Commercial delivery partners who charge per km
- **Admins** — Manage platform, review flagged listings, handle disputes, verify volunteers

### Tech Stack (If Asked "What Did You Build This With?")
- **Frontend**: Next.js 14 App Router, React, TypeScript, Tailwind CSS, shadcn/ui components, Framer Motion animations
- **Backend**: Next.js API Routes (serverless), Prisma ORM
- **Database**: PostgreSQL on Neon Serverless (cold-start handled with retry logic)
- **AI**: Google Gemini 2.5 Flash — used for Vision analysis, bargaining, advisory chat, scout analysis, tag generation
- **Auth**: NextAuth.js with credentials provider, JWT sessions, role-based access control
- **Maps**: Leaflet.js for repair hub locations
- **Charts**: Recharts for dashboard visualizations including custom Sankey diagram

### Scalability (If Asked "Can This Scale?")
- Neon serverless PostgreSQL — auto-scales, pay per query
- Next.js on Vercel — edge functions, auto-scaling
- Gemini API — handles millions of requests
- All agents run async — don't block the user
- City-based partitioning — local-first means less data to query

### Business Model (If Asked "How Would This Make Money?")
- **Free tier** for individuals and NGOs
- **Business subscription** — unlimited listings, ESG reports, priority matching, analytics
- **Transaction fee** — small % on sell transactions (donations are free)
- **Transporter commission** — platform takes a cut from commercial transport bookings
- **Premium AI features** — advanced demand forecasting, bulk listing with AI, custom reports
